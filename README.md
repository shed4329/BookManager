# 图书管理系统

## 简介

这是一个使用mysql+mybatis+JDBC+lombok+JUnit 5+JavaFX(使用FXML，MVC架构)的图书管理系统，当然，只作为练手使用，功能不完全，也不追求完美

> jdk版本:1.8
